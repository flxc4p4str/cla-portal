# cla-portal

Angular SaaS starter generated from a custom template.

## Included

- Angular standalone app structure
- API authentication flow
- JWT local-storage persistence
- auth guard + interceptor
- responsive shell with header + sidenav
- route-driven menu labels
- Signal Forms login page
- API test page using `httpResource`
- merged template dependencies from `package.template.json`
- private npm feed config via `.npmrc`
- `agents.md` and `skills.md`

## Configure the API

Current base URL: `https://absapi.absolution1.com`
